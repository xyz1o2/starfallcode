# 命令提示系统使用指南

## 功能概述

命令提示系统提供实时的命令自动完成和导航功能，帮助用户快速输入命令。

## 使用方式

### 1. 激活命令提示
在输入框中输入 `/` 字符时，命令提示面板会自动显示。

### 2. 导航命令列表

| 快捷键 | 功能 |
|---|---|
| `↑` | 选择上一个命令 |
| `↓` | 选择下一个命令 |
| `Tab` | 自动完成选中的命令 |
| `Esc` | 关闭命令提示 |
| `Enter` | 执行当前输入的命令 |

### 3. 实时过滤

继续输入会自动过滤命令列表。

## 连接到 LM Studio (或其他本地服务器)

你可以使用 `/config-local` 命令来连接任何与 OpenAI API 兼容的本地服务器，例如 LM Studio。

### 步骤

1.  **启动 LM Studio**:
    -   在 LM Studio 中，加载你想要使用的模型。
    -   导航到 "Local Server" 标签页。
    -   点击 "Start Server"。

2.  **获取 Base URL**:
    -   服务器启动后，LM Studio 会显示一个 Base URL，通常是 `http://localhost:1234/v1`。

3.  **在应用中配置**:
    -   使用 `/config-local` 命令，并提供上一步获取的 URL。

    **示例:**
    ```
    /config-local http://localhost:1234/v1
    ```

4.  **(可选) 指定模型**:
    -   如果你的本地服务器需要特定的模型标识符，你可以在 URL 后面提供它。

    **示例:**
    ```
    /config-local http://localhost:1234/v1 my-local-model
    ```

5.  **保存配置**:
    -   为了让这个配置在下次启动时自动加载，请使用 `/save-config` 命令。
    ```
    /save-config
    ```

### 验证

配置完成后，使用 `/status` 命令来检查当前的配置是否正确。
